ABOUT

Commencing with our Token Creation bot, DevKit provides developers with all the utilities required to successfully launch their tokens on any EVM chain. As we continue to roll out, we will continue to introduce more utilities and branch across to more EVM compatible chains in order to provide developers with every opportunity in order to make their project a success.
In the pipeline we already have commenced development on AI logo generator, website templates with automatic population of key token metrics and ability to customize a roadmap. We will also be offering a buy bot, abilities for developers to trend their token based upon the token performance, not based upon payment! Advanced charting tools are under development.

HOW TO USE

Step 1

Press start and then select the create contract option from the menu that pops up

Step 2

Select the blockchain the contract is to be made for and the programming language to be used to write the smart contract

Step 3

Specify the other details such as dev wallets, marketing wallets, airdrop wallets etc

Step 4

Go through your selections and submit when you're okay with it

Step 5

Open the generated text file for the smart contract code
